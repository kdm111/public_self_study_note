# 다차원 배열

### 다차원 배열

다차원 배열의 선언하는 방법은 다음과 같다.

```c
int arr[10];
int arr2[3][4]; // row = 3 col = 4
int arr3[3][3][3]; /// cube
```

이차원 배열 요소의 접근

```c
int arr[3][3];
arr[0][0] = 1;
arr[0][1]= 2; // r=0; c=1에 2대입
```

동시에 초기화하기

```c
int arr[3][3] = {
    {1},
    {4,5},
    {7,8,9}
};
```

다음과 같은 방법으로 초기화 할 수 있다.
