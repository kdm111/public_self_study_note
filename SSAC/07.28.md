기본 for문 사용

```js
for (let i = 0; i <arr.length; i++) {
  console.log(arr[i])
}
```

```js
for (let item of arr) {
  console.log(item)
}
```

```js
arr.forEach((item, i, arr) {
	console.log(item, i, arr)
})
```
